### Command: mkdir dirA dirB
Creates two directories named dirA and dirB inside the current folder.

### Command: echo "text" > filename
Creates a new file (filename) and writes "text" into it. If the file exists, it overwrites the content.

### Command: ls dirA / ls dirB
Lists all files in the specified directory.

### Command: nano sync.sh
opens nano text editor 

### Command: chmod +x sync.sh
Creates exectuable file

### Command: ./sync.sh dirA dirB
Executes the sync.sh script with dirA and dirB as arguments.
